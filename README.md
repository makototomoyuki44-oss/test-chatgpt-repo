# test-chatgpt-repo

Python の練習用リポジトリです。

簡単な関数の練習から、コンソールで遊べるゲーム、画面付きのゲームまで入っています。

## ファイル一覧

| ファイル名 | 内容 | 実行方法 |
| --- | --- | --- |
| `python_test.py` | 2つの数字を足す `add` 関数の練習コードです。実行すると `2 + 3` の結果として `5` を表示します。 | `python python_test.py` |
| `python_test2.py` | コンソールで遊べるオセロ（リバーシ）ゲームです。人間とコンピュータが対戦できます。 | `python python_test2.py` |
| `python_test3.py` | `tkinter` を使ったブロック崩しゲームです。マウスや左右キーでパドルを操作します。 | `python python_test3.py` |
| `test.py` | 空の Python ファイルです。これから練習コードを書くためのファイルとして使えます。 | 中身を追加してから `python test.py` |
| `test_test.txt` | GitHub にテキストファイルを置く練習用のファイルです。 | 実行するファイルではありません |
| `.gitignore` | Python の一時ファイルや仮想環境、エディタ設定などを GitHub に上げないための設定ファイルです。 | 実行するファイルではありません |

## 実行に必要なもの

Python がインストールされていれば、基本的に実行できます。

`python_test3.py` は Python 標準ライブラリの `tkinter` を使います。環境によっては `tkinter` が別途必要になる場合があります。

## 実行例

```bash
python python_test.py
```

出力例:

```text
5
```

## 初心者向けメモ

- `python_test.py` は、関数の基本を学ぶための短いコードです。
- `python_test2.py` は、条件分岐、ループ、リスト、関数分割の練習になります。
- `python_test3.py` は、クラス、画面描画、キーボードやマウス入力の練習になります。

まずは `python_test.py` を読んでから、少しずつ `python_test2.py`、`python_test3.py` に進むのがおすすめです。
